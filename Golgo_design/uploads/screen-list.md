# Golgo (고르고) — Claude Design 전달용 화면 목록

> 서비스명: Golgo (고르고)
> 플랫폼: PWA (Web + 모바일 홈화면 설치)
> UI 라이브러리: shadcn/ui + Tailwind CSS + Lucide React
> 타겟 유저: 주식 초보자 · 20~40대
> 톤앤매너: 친근하고 신뢰감 있는, 복잡하지 않은
> 레퍼런스 앱: 토스, Robinhood

---

## 화면 구조 개요

```
인증
├── 로그인
└── 닉네임 설정 (최초 1회)

온보딩 (최초 1회, 4단계)
├── 1단계: 닉네임 설정
├── 2단계: 투자 성향 선택
├── 3단계: 증권사 연결
│   ├── 3-1: 증권사 선택
│   ├── 3-2a: KIS 연동 방식 선택 (API Key / 캡처)
│   ├── 3-2b: 캡처 업로드
│   ├── 3-2c: 파싱 결과 편집
│   └── 3-2d: 종목 수정 바텀시트
└── 4단계: 목표 비중 설정 (선택)

메인 (바텀 네비 4탭)
├── 탭1: 홈 대시보드
├── 탭2: 포트폴리오
│   ├── 포트폴리오 상세
│   └── 캡처 업로드 (최신화)
├── 탭3: AI 분석
│   ├── 분석 목록
│   └── 분석 리포트 상세
└── 탭4: 리밸런싱
    └── 리밸런싱 가이드

설정
├── 계좌 관리
├── 전략 설정
├── 알림 설정
└── 프로필 (소셜 계정 관리)
```

## 디자인 요청 사항

1. **컬러 시스템**: 투자 도메인에 맞는 신뢰감 있는 팔레트 (과도한 원색 지양)
2. **타이포그래피**: 숫자(금액·수익률)가 명확히 읽히는 폰트 선택
3. **상태 색상 통일**:
   - 수익 / 매수 / 정상 → 초록 계열
   - 손실 / 매도 / 오류 → 빨강 계열
   - 경고 / outdated → 주황 계열
   - 유지 / 중립 → 회색 계열
4. **모바일 우선**: 390px 기준 설계, 바텀 네비게이션 고정
5. **다크모드**: 선택 구현 (우선순위 낮음)
