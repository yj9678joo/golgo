# Golgo (고르고) — 사람 + AI Agent 협업 개발 로드맵

> 기준일: 2026년 5월 14일
> 개발 방식: 1인 개발 + AI Agent (Codex · Cursor AI) 협업
> 운영 환경: Vercel + Railway + Supabase — 무료티어

---

## 협업 방식 정의

### AI Agent 도구

| 도구 | 용도 |
|------|------|
| **Codex** | 터미널 기반 코드 생성·수정·리팩토링 |
| **Cursor AI** | IDE 내 실시간 코드 보완·자동완성 |
| **Claude** | 설계 검토·프롬프트 작성·의사결정 보조 |

### 역할 분담 원칙

```
🧑 사람    → 설계 확정, 의사결정, 보안 검토, 외부 API 연동 판단, 최종 QA
🤖 AI Agent → 코드 생성, 보일러플레이트, 테스트 작성, 컴포넌트 구현, 리팩토링
🤝 협업    → LLM 프롬프트 튜닝, 복잡한 비즈니스 로직, 디버깅
```

### 표기 방식

- 🧑 사람이 주도
- 🤖 AI Agent가 주도
- 🤝 협업 (사람이 방향 제시 → AI가 구현 → 사람이 검토)
